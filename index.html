Projects

GCN4EPI

Designed a multimodal graph convolutional network model to identify gene regulatory elements. Integrated DNA sequence and Enhancer-Promoter interaction data. Achieved +10% accuracy and published in Computational Biology and Chemistry.

Python, PyTorch, TensorFlow, Bash, Git

CoV-SNN

Developed a lightweight protein language model and a contrastive learning framework for viral escape prediction. Achieved 97% accuracy and 125× speedup. Funded by the Council of Higher Education of Türkiye and published in IEEE TCBB.

Python, PyTorch, Streamlit, HTML, CSS, JS, Git

CodeSizeBERT

Constructed manually labeled datasets and fine-tuned a transformer model for automated software size measurement from Python code using COSMIC and MicroM standards. Published in Automated Software Engineering.

Python, PyTorch, Transformers, BERT, Git

GO-cluster

Designed weighted gene co-expression networks integrating protein-protein interaction, gene expression, and gene ontology data. Identified 22 shared genes in three metabolic disorders. Published the method and results in IEEE TCBB.

R, Python, Bash, Git, GOSemSim, STRINGdb

FastSbR

Developed global optimization algorithms for an NP-hard problem: Sorting unsigned genomes by reversals. Achieved 5.6-fold speedup compared to baselines. Supported by the Scientific and Technological Research Council of Türkiye.

C, MPI, OpenMP, Bash, SLURM, Git

AI-Estimator

Designed and deployed task-specific BERT models for automated software analytics. Conducted industrial case studies. Funded by the Scientific and Technological Research Council of Türkiye and published in Journal of Systems and Software.

Python, PyTorch, Google Vertex AI, HTML, CSS, JS, Git, Docker

ARtNet

Developed a graph-based artist recommendation framework that integrates association rule mining with community detection. Constructed Neo4j graph databases to identify artist clusters on Spotify's playlist data. Presented in KDIR 2021.

Python, Neo4j, scikit-learn, Apyori, NetworkX

DentAI

Used graph convolutional networks for 3D dental image segmentation and modeling. Led software development and management activities. Coordinated a team of AI engineers and dentists. Collaborated with Ege University, Faculty of Dentistry.

Python, PyTorch, HTML, CSS, JS, Git, Blender

MorphoMark

Built a deep learning model for 3D facial landmark detection on mesh and point cloud data. Enabled automated anatomical point localization and geometric measurement for skeletal classification in orthodontic and craniofacial analysis.

Python, PyTorch, Open3D, NumPy, MeshLab

OptiMine

Built an intelligent agent that integrates multimodal sensor data (vibration, pressure, temperature, throughput) to monitor operations, predict equipment failures, optimize process parameters, and improve efficiency in mining and mineral processing plants.

Python, LangChain, XGBoost, PostgreSQL, MLflow, Streamlit
